# /rule

## 别名

* /new

## 解释

创建一条规则。

## 用法

用法 1：

```
/rule <订阅链接>
```

用法 2：

```
/rule <订阅链接> <规则名>
```

`<规则名>` 是可选的，当不提供时，会自动生成规则。

创建好规则后，你就可以使用以下操作进行测试，而不用每次都输入订阅链接：

```
/test <规则名>
```

用法 3：

```
/rule <规则名>
```

如果规则名存在，就会查看具体的规则设置。

## 特性

> * `/rule` 无参数将显示自己创建的规则
> * 管理员可以看到所有规则名称及背后的配置，并使用它们
> * 规则名称是唯一的，不是自己创建的规则无法使用，也就意味着你无法测试别人规则里包含的订阅
> * 自己创建的规则可以覆写
> * 目前仅适配了 `/re` 以及测试指令
> * `/rule` 如果不写规则名，将随机生成一个规则名
> * 可以使用 `/remove` 删除规则
