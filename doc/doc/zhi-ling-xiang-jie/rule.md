# /rule

## 别名

* /new

## 解释

创建一条规则

## 用法

用法1：

```
/rule <订阅链接>
```

用法2：

```
/rule <订阅链接> <规则名>
```

<规则名> 是可选的，当不提供时，将会自动生成规则。

创建好规则后，你就可以使用以下操作进行测试，而不用每次都输入订阅链接：

```
/test <规则名> 
```

用法3：

```
/rule <规则名> 
```

如果规则名存在，将查看具体的规则设置。

## 特性

> * /rule 无参数将显示自己创建的规则&#x20;
> * 管理员可以看到所有规则名称及背后的配置，并使用它
> * 规则名称是唯一的，不是自己创建的规则无法使用，意味着你无法测试别人规则里面包含的订阅。
> * 自己创建的规则可以覆写&#x20;
> * 目前仅适配了 /re 以及 测试指令&#x20;
> * /rule 如果不写规则名，将随机生成一个规则名&#x20;
> * 可以使用  /remove 删除规则
